# 中国出发：土耳其 + 埃及 14 天旅游计划

## 先说结论

这版按“第一次去土耳其 + 埃及、标准体力、从中国主要城市出发、未指定酒店和日期”来排。路线是：

中国 → 伊斯坦布尔 4 天 → 卡帕多奇亚 3 天 → 开罗 3 天 → 卢克索 2 天 → 开罗 1 天缓冲 → 中国

核心取舍：14 天不再塞棉花堡、以弗所、安塔利亚、阿斯旺、阿布辛贝、亚历山大和红海。双国旅行最怕转场过多，这些点都值得去，但会把行程变成赶路。

## 重要假设

- 出发城市：中国任一主要国际口岸，具体航班按北京/上海/广州/深圳/成都等替换。
- 日期：未指定，所以不绑定节假日、天气和具体航班时刻。
- 住宿：伊斯坦布尔住 Sultanahmet 或 Karakoy/Galata；卡帕多奇亚住 Goreme；开罗住 Giza 或 Zamalek；卢克索住东岸。
- 节奏：每天一个主区域，第一天和最后一天轻量，热气球预留两天容错。
- 预算：不含国际机票，舒适型大约 USD 1500-2300/人；热气球、酒店档次和埃及包车会显著影响总价。

## 每天怎么走

### Day 1：中国 → 伊斯坦布尔

落地后直接进城，住 Sultanahmet。只安排酒店附近晚餐，不进景区。

目的：长途飞行后先恢复，第二天步行覆盖老城核心。

### Day 2：伊斯坦布尔老城

圣索菲亚 → 蓝色清真寺 → 地下水宫 → Sultanahmet 午餐 → 托普卡帕宫 → 大巴扎

这是伊斯坦布尔最核心的一天。托普卡帕宫是当天唯一重票务点，不再硬塞多个大型博物馆。

### Day 3：博斯普鲁斯海峡 + Galata

Eminonu 码头 → 博斯普鲁斯短线游船 → Karakoy 午餐 → Galata Tower → Istiklal Avenue → Galata/Karakoy 晚餐

天气好时坐船，天气差时把游船替换为室内博物馆或咖啡馆。

### Day 4：Dolmabahce + Kadikoy 亚洲岸

Dolmabahce Palace → Ortakoy → 午餐 → 渡轮去 Kadikoy → Moda 海边 → Kadikoy 晚餐

这天比前两天慢一点，补一个生活化街区，避免只看大景点。

### Day 5：伊斯坦布尔 → 卡帕多奇亚

飞 Nevsehir 或 Kayseri → 接驳到 Goreme → 入住洞穴酒店 → Goreme 日落点 → Goreme 晚餐

转场日只安排日落，不做长距离徒步。

### Day 6：卡帕多奇亚热气球 + Goreme 经典线

热气球 → Goreme Open Air Museum → 午休 → Uchisar Castle → Love Valley Viewpoint

热气球高度依赖天气，所以 Day 6 和 Day 7 都留有容错。如果 Day 6 取消，顺延 Day 7。

### Day 7：地下城 + 山谷缓冲

Kaymakli Underground City → Pigeon Valley → Goreme 午休 → Pasabag → Devrent Valley

如果热气球顺延到这天，下午只保留 Pasabag，其他删掉。

### Day 8：卡帕多奇亚 → 开罗

Goreme → 机场 → 伊斯坦布尔转机 → 开罗 → 住 Giza/Zamalek → 轻晚餐

跨国转场日不进老城，不去金字塔，先处理入境、现金和休息。

### Day 9：Giza 金字塔 + 大埃及博物馆

Giza Pyramid Complex → Sphinx → Giza 午餐 → Grand Egyptian Museum → Giza 日落/晚餐

金字塔和 GEM 在同一大区域，是开罗最顺的一天。上午户外，下午室内。

### Day 10：伊斯兰开罗 + 市区

Citadel of Saladin → Muhammad Ali Mosque → Khan el-Khalili 午餐/咖啡 → 市集 → 尼罗河/Zamalek 晚餐

如果你更想看文物，可以把下午市集替换为 Tahrir 的 Egyptian Museum，但不要同一天硬塞 GEM + 老馆 + 老城。

### Day 11：开罗 → 卢克索东岸

早班机飞卢克索 → 入住东岸 → Karnak Temple → Luxor Temple 夜景 → 河边晚餐

双国 14 天里建议飞，不建议用夜车省钱，因为体力成本很高。

### Day 12：卢克索西岸

Valley of the Kings → Temple of Hatshepsut → Colossi of Memnon → 西岸午餐 → 傍晚尼罗河帆船或休息

这是埃及段最重要的古迹日。清晨出发，避开中午高温。

### Day 13：卢克索 → 开罗缓冲

上午补拍/休息 → 下午飞回开罗 → 住机场或 Heliopolis → 轻晚餐

这一天故意留缓冲，避免最后一天从卢克索直接衔接国际航班造成误机风险。

### Day 14：开罗 → 中国

按航班去机场。若是深夜航班，白天只安排酒店周边轻活动或商场，不再跑金字塔区、老城或远郊。

## 每天吃什么

- 伊斯坦布尔老城：Sultanahmet 周边土耳其烤肉、pide、家常菜，先看距离和评价，不追网红。
- Karakoy/Galata：海鲜、小酒馆、屋顶餐厅都可，适合晚餐和夜景。
- Kadikoy：可搜 Ciya Sofrasi 周边或市场餐厅，适合尝安纳托利亚菜。
- Goreme：陶罐牛肉、当地土耳其菜，热气球前一晚别喝太多。
- 开罗：Giza 选金字塔景观餐厅；Zamalek/尼罗河边更适合城市晚餐。
- 卢克索：东岸河边餐厅最方便，西岸午餐以顺路为主。

餐厅原则：按当天主区域选，不为了某家餐厅跨城折返。地图页里每个餐饮点都放了大众点评和小红书搜索入口，可到出发前再精修。

## 天气敏感点

- 博斯普鲁斯游船：大风、雨天、能见度差时替换为室内点。
- 卡帕多奇亚热气球：强天气敏感，至少预留两个清晨。
- 卡帕多奇亚峡谷：风沙或暴晒时缩短徒步。
- Giza 金字塔和卢克索西岸：强日晒，尽量清晨出发。
- 尼罗河帆船：风大或时间不稳时直接取消。

## 提前订什么

- 土耳其境内机票：伊斯坦布尔 ↔ 卡帕多奇亚。
- 卡帕多奇亚热气球：越早订越稳，但要确认取消和改期政策。
- 土耳其博物馆/景点门票：托普卡帕宫、Galata Tower、Goreme Open Air Museum 等出发前查 muze.gov.tr。
- 埃及签证：出发前在 Egypt e-Visa 官方门户按护照信息核验。
- GEM 和重点古迹：出发前查官方票务与开放时间。
- 埃及内陆机票：开罗 ↔ 卢克索。
- 卢克索西岸包车或靠谱向导：Day 12 会省很多沟通成本。

## 删掉什么

- 土耳其：棉花堡、以弗所、安塔利亚。原因是会增加至少 2-4 天或多次飞行/长途车。
- 埃及：阿斯旺、阿布辛贝、红海、亚历山大。原因是 14 天双国路线已经有开罗和卢克索两个重古迹区，再塞会牺牲休息和容错。
- 约旦/佩特拉：完全不放。原因是三国路线需要重新设计，不适合这版“土耳其 + 埃及 14 天”。
